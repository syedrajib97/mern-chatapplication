## Run Locally

Clone the project

```bash
  git clone https://github.com/syedrajib97/mern-chatapplication.git
```

Go to the project directory

```bash
  cd MERN-CHAT-APP
```

Install dependencies

```bash
  cd client/
  npm install
```

```bash
  cd server/
  npm install
```

Start the server

```bash
  cd server/
  npm run dev
```
Start the Client

```bash
  cd client/
  npm start
```
